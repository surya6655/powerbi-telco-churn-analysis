# powerbi-telco-churn-analysis
Power BI dashboard analyzing telecom customer churn behavior using customer demographics, contracts, tenure and payment methods.
📊 Telco Customer Churn Analysis – Power BI Dashboard
Project Overview

This project analyzes telecom customer churn behavior using Power BI to identify the key factors contributing to customer attrition. The dashboard provides interactive insights into customer demographics, contract types, payment methods, tenure groups, and revenue impact.

The goal of this project is to help businesses identify high-risk customer segments and implement data-driven strategies to reduce churn and improve retention.

📈 Dashboard Pages
1️⃣ Churn Overview Dashboard

This page provides a high-level summary of telecom customer churn metrics.

Key Visualizations

Total Customers KPI

Churned Customers KPI

Churn Rate Percentage

Revenue Overview

Churn % by Senior Citizen

Customers by Payment Method (Churn vs Retained)

Churn by Partner & Dependents

Customer Tenure vs Churn Trend

Contract Distribution Analysis

Key Insights

Customers with Month-to-Month contracts have the highest churn rate

Customers paying through Electronic Check show higher churn behavior

Customers with short tenure (0–6 months) churn more frequently

Long-term contract customers tend to have lower churn probability

These insights help organizations identify high-risk churn segments.

2️⃣ Customer Details – Drill Through Page

This page provides customer-level analysis through drill-through functionality.

Users can drill into detailed customer information directly from the overview dashboard.

Available Customer Attributes

Tenure Duration

Monthly Charges

Total Charges

Contract Type

Payment Method

Internet Service

Senior Citizen Label

Dependents Label

Churn Status

Purpose

This page enables analysts and business teams to investigate churn patterns at the individual customer level.

3️⃣ Churn Analysis – Decomposition Tree

This page uses Power BI’s Decomposition Tree visual to identify the root causes of churn.

Users can drill down through multiple business dimensions such as:

Senior Citizen

Contract Type

Payment Method

Tenure Group

Partner Status

Dependents

Key Findings

The decomposition analysis shows that churn is strongly associated with:

Month-to-Month contracts

Electronic Check payment method

Customers with lower tenure periods

This analysis helps businesses identify drivers of churn and optimize retention strategies.
